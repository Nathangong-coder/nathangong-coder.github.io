Before we start, every link has some basic things it requires. It requries the "a" tag and something inside the tag for the link to show up.

That should look like this:

                
                    <a>Cool Link</a>

First, we'll talk about the basic link, linking to another page

All you need for that is the "a" tag and a website href (as shown below):

                
                    <a href="https://css-tricks.com" CSS-Tricks> </a>

Secondly, a link can to another page, like for a navbar

All you need for that is the "a" tag and that page's href:

                
                    <a href="index.html"> Home </a>

You can also make a jump link, for when you want to link to part of your webpage

All you need for that is the "a" tag and that element's id href: 

                
                    <h1 id="#top"> Hello! Welcome to my demo! </h1>

                    ....

                    <a href="#top">Back to top</a>

You can also make disabled links!

These look a bit weird, like this:

Weird Disabled Links

These are mainly used if a link may be deactivated until somebody presses/clickes on something, and have no href tag, like this:

                
                    <a>Weird Disabled Links</a>

How about opening links in new tabs?

This is generally unrecommended, as you will also need rel attributes (shown later). It requires target="_blank" to work, and is done like this: 

                
                    <a href="www.css-tricks.com" target="_blank"> </a>

What are some rel attributes, which do I need for target="_blank", and how do I use them?

There are many, some include:

  • rel="alternate" (Alternate version of the document)
  • rel="author" (Author of the document)
  • rel="help" (A resource for help with the document)
  • rel="search" (A document meant to perform a search in the current document)
  • rel="sponsored" (Informs search engine that a marked link/page is sponsored)
  • rel="ugc" (Informs search engine of not-particularly-trusted user-generated content, like comments and forum posts)
  • rel="nofollow" (Informs search engine that linked site is not associated with current site)
  • rel="noopener" (Prevents the page from doing malicious things)
  • rel="noreferrer" (Prevents other sites from identifying how the user got to the clicked on link)
  • rel="me" (Indicates that its destination represents the same person or entity as the current page)

It is simple to implement, and add a space in between two rels to separate them like this:

                
                    <a href="www.youtube.com" target="_blank" rel="sponsored noopener noreferrer"></a>
